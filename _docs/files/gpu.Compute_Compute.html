Interface Compute
access this type via: gpu.Compute (provides, requires or uses)
This interface allows the user to interact with compute devices available to the system

Constants
FLOAT Flag to indicate to used the float type on the compute device. What 'float' means precicly depends on the implimenting component and what compute device is being used
UINT Flag to indicate to used the unsigned integer type on the compute device. What 'unsigned integer' means precicly depends on the implimenting component and what compute device is being used
void init_w()
Initlises whatever API is being used in the native library to communicated with the compute devices
void findPlatforms_w()
Asks the native library what compute platforms are supported. For example CUDA or Intel Iris.
int[] getComputeDeviceIDs_w()
Asks the native libraray to return all the raw handles to all the available compute devices on the system
String[] getComputeDevices_w()
Asks the native library to return all the names of the compute devices available on the system
void createContext_w(int devices[])
Asks the native library to create some 'state' that contains all the devices given by the integer array parameter
int createAsynchQueue_w(int device)
Asks the native library to create a asynchronous queue that points to the compute device given by 'device'. These queues are used to request read/write and execution commands to the compute device. Returns a native library handle to that Queue
int createSynchQueue_w(int device)
Asks the native library to create a synchronous queue that points to the compute device given by 'device'. These queues are used to request read/write and execution commands to the compute device. Returns a native libraray handle to that Queue
int createArray_w(int device, int length, int type)
Asks the native library to create a 1-D buffer of 'length' size and of the type given by 'type' on the compute device given by 'device'. Returns a native library handle to that buffer.
int createMatrix_w(int device, int width, int height, int type)
Asks the native library to create a 2-D buffer of 'width' * 'height' size and of the type given by 'type' on the compute device given by 'device'. Returns a native library handle to that buffer.
void destroyMemoryArea_w(int memObj_cl)
Asks the native library to deallocate the memory on a compute device pointed to by 'memObj_cl'
int[] createProgram_w(char source[])
Asks the native library to compile the source code given by 'source' for all platforms/compute devices available to the system. Returns an array of native library handles to the compiled programs.
void writeIntArray_w(int queue_cl, int memObj_cl, int values[])
Asks the native library to write to the buffer given by 'memObj_cl' the values given by 'values' through the queue given by 'queue_cl'.
int[] readIntArray_w(int queue_cl, int memObj_cl, int buf[])
Asks the native library to read the values in the buffer (up to the length of the 'buf' array) given by 'memObj_cl' through the queue given by 'queue_cl' and return the values to the caller
void writeFloatArray_w(int queue_cl, int memObj_cl, dec values[])
Asks the native library to write to the buffer given by 'memObj_cl' the values given by 'values' through the queue given by 'queue_cl'.
dec[] readFloatArray_w(int queue_cl, int memObj_cl, int len)
Asks the native library to read the values (up to the index given by 'len') in the buffer given by 'memObj_cl' through the queue given by 'queue_cl' and return the values to the caller
void writeIntMatrix_w(int queue_cl, int memObj_cl, int buf[][])
Asks the native library to write to the buffer given by 'memObj_cl' the values given by 'buf' through the queue given by 'queue_cl'.
int[][] readIntMatrix_w(int queue_cl, int memObj_cl, int buf[][])
Asks the native library to read the values in the buffer (up to the width given by buf[0] and height given by buf[1]) given by 'memObj_cl' through the queue given by 'queue_cl' and return the values to the caller
void writeFloatMatrix_w(int queue_cl, int memObj_cl, dec buf[][])
Asks the native library to write to the buffer given by 'memObj_cl' the values given by 'buf' through the queue given by 'queue_cl'.
dec[][] readFloatMatrix_w(int queue_cl, int memObj_cl, dec buf[][])
Asks the native library to read the values in the buffer (up to the width given by buf[0] and height given by buf[1]) given by 'memObj_cl' through the queue given by 'queue_cl' and return the values to the caller
int prepareKernel_w(int clProghandle, int paramHandles[], int paramCount, char program[])
Asks the native library to prepare the program given by 'clProghandle' to be executed with the parameters given by 'paramHandles'. 'program' is a char array that gives the name of the program, which should match the function name given to the kernel in the source code provided in the createProgram_w function of this interface.
void runKernel_w(int clKernel, int queue_cl, int outputDims[])
Asks the native library to execute the kernel given by 'clKernel' through the queue given by 'queue_cl' and execute it on a set of threads with dimention and size outputDims[0] * outputDims[0+i] * ..., * outputDims[n]
