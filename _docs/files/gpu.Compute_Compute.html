Interface Compute
access this type via: gpu.Compute (provides, requires or uses)
This interface allows the user to interact with compute devices available to the system

Compute(char device[], storeComputeArray computeArr)
Create the instance to interact with the device identified by the name 'device'. The computeArr is how this compute device is binded to the native API
char[] getPlatform()
Indentifies which compute platform this device belongs to, e.g. CUDA
char[] getDevice()
Indentifies the device by name
void runProgram(Program p)
Runs program p on this compute device
