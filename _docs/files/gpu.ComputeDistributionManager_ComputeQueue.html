Data type ComputeQueue
access this type via: gpu.ComputeDistributionManager (uses)
A container that has all the information needed to call the functions related to read/write and excute commands in the gpu.Compute interface. It also contained a Dana Queue instance to provide to other components that may wish to use the queue without exposing native library handles to them.
Fields
int queue_cl Native library handle that identifies a queue
Queue q Dana Queue optionally used to allow other components to interact with the native queue indirectly
String physicalDevice The name of the device this queue is associated with
int physicalDeviceID The native library handle of the device this queue is associated with
