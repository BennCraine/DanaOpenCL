Builds the program from the source code provided for the Compute device given. functionName must match the kernel function name in the source code.
void setParameters(ExtMemory parameters[])
The entry point function of the provided source code for this program will list a set of parameters. Representations of these parameters are given by the types in this package that extend the ExtMemory type. Arrange these in an array and pass it to this funciton to have those parameters used when this Program object is executed
